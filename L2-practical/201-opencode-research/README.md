## 📊 核心要点

### OpenCode 是什么？

OpenCode 是一个开源的 AI 智能体框架，提供：
- 🤖 **Agent 开发**：快速构建 AI 智能体
- 🔌 **插件系统**：丰富的扩展能力
- 🌐 **Web 集成**：与浏览器深度集成
- 📦 **SDK 支持**：多语言 SDK

### 核心优势

| 特性 | 说明 |
|------|------|
| 轻量级 | 核心代码精简 |
| 易扩展 | 插件化架构 |
| 社区活跃 | 持续更新 |
| 文档完善 | 丰富的教程 |
