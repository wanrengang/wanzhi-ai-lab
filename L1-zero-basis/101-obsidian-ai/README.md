## 🚀 快速开始

### 安装 Obsidian

1. 访问 [obsidian.md](https://obsidian.md)
2. 下载并安装
3. 创建第一个知识库

### 基础操作

```markdown
# 创建笔记
点击"新建笔记"

# 添加双向链接
输入 [[笔记名称]]

# 添加标签
输入 #标签名
```

### AI 插件推荐

- 🔌 **Text Generator**: AI 文本生成
- 🔌 **Smart Connections**: AI 知识关联
- 🔌 **Copilot**: AI 写作助手
