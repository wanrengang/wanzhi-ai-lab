## 🚀 快速开始

### 我是零基础新手

```bash
# 推荐：从 L1 工具教程开始
cd L1-zero-basis/202-no-code-agent/

# 查看教程
cat README.md

# 跟着步骤操作即可
```

**学习路径**：
1. 看 [AI Agent 是什么？](./L1-zero-basis/101-ai-agent-intro/)
2. 跟着 [5分钟搭建 AI 助手](./L1-zero-basis/202-no-code-agent/)
3. 尝试 [用 AI 自动写日报](./L1-zero-basis/203-auto-daily-report/)

### 我有 Python 基础

```bash
# 推荐：从 L2 实战项目开始
cd L2-practical/401-knowledge-base/

# 安装依赖
pip install -r requirements.txt

# 配置环境
cp .env.example .env

# 运行项目
python main.py
```

**学习路径**：
1. 选择 [实战项目](#-l2---进阶实战)
2. 阅读 [框架对比](#-l2---进阶实战)
3. 查看 [避坑指南](#-l2---进阶实战)

### 我是专业开发者

```bash
# 推荐：直接看 L3 深度技术
cd L3-advanced/801-langchain-source/

# 阅读源码分析
cat SOURCE_CODE_ANALYSIS.md
```

**学习路径**：
1. 阅读 [源码分析](#-l3---专业深度)
